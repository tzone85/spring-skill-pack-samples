- [`spring-security-jwt-resource-server`](skills/spring-security-jwt-resource-server.md) — Spring Security 6.x DSL, scope + role conversion, MockMvc test patterns.
- [`hexagonal-architecture-spring-boot`](skills/hexagonal-architecture-spring-boot.md) — multi-module split, domain stays framework-free, ArchUnit guards the boundary.
- [`flyway-migrations-zero-downtime`](skills/flyway-migrations-zero-downtime.md) — safe/unsafe matrix, three-release rename pattern, CI gate template.

## How to use

Drop a `.md` file into `.claude/skills/` in your project (or `~/.claude/skills/` for user-level). Claude Code picks it up automatically.
